Module Content¶

Each module you take will consist of one or more sections. These sections can cover the full range of content, including lectures, tutorials, exercises,… You can access these sections by clicking on the module’s title in the list of your modules. Then select the section you want to view by clicking on it’s title. Currently there are two different types of sections:

  • Text Sections
  • HTML Editor Sections
