Module¶

The module is the main high-level structuring element in the Web Teaching Environment. It contains a number of parts that divide the module’s content into multiple sections. In addition to this a module also defines access settings that specify restrictions on which learners can take the module and you can then also manage those learners (or tutors) that are taking the module.

The following actions are available on modules via the menu icon, both on the module overview page and also for each module on the list of your modules:
