wte.scripts.database – Database scripts¶

The database module provides the functionality for creating the initial database.

wte.scripts.database.init(subparsers)[source]¶

Initialises the ArgumentParser, adding the “initialise-database” command.

wte.scripts.database.initialise_database(args)[source]¶

Initialises the database schema and adds the default Permission, PermissionGroup, and User to the database.
