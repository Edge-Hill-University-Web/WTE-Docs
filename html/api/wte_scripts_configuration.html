wte.scripts.configuration – Generate configuration script¶

The configuration module provides the functionality for generating a configuration file from the default template and generating new styles.

wte.scripts.configuration.generate_config(args)[source]¶

Generates a configuration file based on the default_config.txt template.

wte.scripts.configuration.init(subparsers)[source]¶

Initialises the ArgumentParser, adding the “generate-config” and “generate-custom-styling” commands.
